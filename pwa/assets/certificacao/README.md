# Selos de certificação — PDF proposta comercial

O ficheiro `selos-certificacao.png` é desenhado no **canto inferior direito** de todas as páginas do PDF MS.015.

Para substituir os selos, mantenha o nome do ficheiro ou atualize `ORCAMENTO_CERTIFICACAO_SELOS_SRC` em `pwa/js/pdf-orcamento-certificacao.js`.

Recomenda-se PNG com fundo transparente e faixa horizontal (ISO 9001, PME Líder, PME Excelência, etc.).
